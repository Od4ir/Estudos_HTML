Principais Formatações

Negrito / Destaque

Vmaos colocar uma palavra em negrito utilizando a tag <b> (Mas não tem semântica).

Porém, utilizando a tag <strong> temos um negrito sentido semântico.

Itálico / Ênfase

Vamos escrever uma parte deste texto em itálico para vermos como fica.

Lembrando que este itálico feito com a tag <i> não é semântico.

Já nesta frase temos um texto em itálico utilizando a tag <em> (com semântica).
